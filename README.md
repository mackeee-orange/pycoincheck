
# pycoincheck

*pycoincheck* is a python wrapper of CoinCheck's REST API.
<https://coincheck.com>

This is [Document][]

## Install
Using pip
`$ pip install pycoincheck`

## Usage
    import pycoincheck
    api = pycoincheck.API(api_key="your api_key", api_secret="your api_secret")
